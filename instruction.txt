# 🌿 Запуск проекта с нуля

1. переименуй `database.example.json` на `database.json`.
2. Установи библиотеки (в терминале папки проекта):
   npm install
3. Запусти сервер в терминале:
   node server.js
4. Открой в браузере:
   http://localhost:3000

⚠️ Важно:
- Через Live Server (порт 5500) POST-запросы работать НЕ будут. Запуск только через localhost:3000.
- При изменении server.js перезапусти сервер в терминале (Ctrl+C -> node server.js).
- Если мешают расширения Chrome, тестируй в режиме Инкогнито.
