# Что добавлено и как это запустить

Ничего переустанавливать не нужно — новый код работает поверх вашего `express` и `database.json`. Просто замените файлы проекта на эти и запустите как обычно: `node server.js`.

## 1. Смена пользователя
- В сайдбаре сверху появился блок с именем текущего пользователя и кнопкой «🔄 Сменить пользователя».
- Нажатие очищает `localStorage` и перезагружает страницу — снова появится окно входа (отдельного `login.html` в проекте нет, вход всегда был модальным окном поверх `index.html`).

## 2. Автозаполнение справочника из Trefle API
1. Зарегистрируйтесь бесплатно на https://trefle.io и получите личный токен (общего "публичного" ключа не существует).
2. Откройте `seed-config.js`, вставьте свой токен вместо `ВСТАВЬТЕ_СЮДА_ВАШ_КЛЮЧ_С_TREFLE.IO`.
3. Запустите: `node seed-plants.js`
4. Скрипт добавит новые растения в `database.json` (поле `catalog`) и обновит уже загруженные ранее (по `trefleId`) — сколько угодно раз перезапускать безопасно.
5. ⚠️ Trefle не даёт точных данных о поливе/пересадке/токсичности — эти поля заполняются общими формулировками на основе шкал освещённости и влажности. При желании отредактируйте вручную в `database.json`.
6. Нужен Node.js 18+ (используется встроенный `fetch`, без новых npm-пакетов).

## 3. Чат по растениям
- Кнопка «💬 Чат» на карточке в справочнике — открывает обсуждение конкретного растения.
- Пункт «💬 Обсуждения» в сайдбаре — список всех растений для выбора чата.
- Сообщения хранятся в `database.json` → `chats[plantId]`.
- Обновление раз в 5 секунд через `setInterval`, без WebSockets, как и просили.
- Автообновление останавливается при уходе со страницы чата (чтобы не грузить сервер зря).

## 4. Поиск в справочнике
- Строка поиска над сеткой растений на странице «Справочник», фильтрует по названию.

## 5. Кнопка «Полил!»
- Уже была реализована в исходном проекте (`collection.js`, кнопка «Полил(а)» обновляет `lastWatered` и сохраняет на сервер) — ничего добавлять не потребовалось.
